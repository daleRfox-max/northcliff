# Task 05 — Stronger primary CTA copy

**Tier:** 1
**Type:** EDIT

## Problem
"Inquire About Leasing" is generic and feels like a commitment. For a project opening Dec 2027, prospects aren't ready to lease — they're ready to *watch and wait*. The CTA should match where they are in the journey.

## Goal
Replace every instance of "Inquire About Leasing" with softer-commitment copy that converts better at this stage.

## Recommended primary CTA
**Join the Priority Leasing List**

## Alternatives (pick one and use consistently)
- Get Availability Updates
- Reserve Your Place on the List
- Stay in the Loop
- Be First to Know

## Where to update
- Hero section primary CTA
- Navigation bar (if "Inquire" appears there)
- Leasing section CTA
- Sticky mobile bar (Task 03)
- Footer CTA if applicable
- `/contact.html` page submit button (might say "Send Inquiry" — update to match)

## Tone guidance
The full inquiry form (longer fields, more commitment) can keep slightly stronger language like "Request Information" or "Contact Jennifer." The waitlist/short form should use the soft language above.

## Acceptance criteria
- [ ] No instances of "Inquire About Leasing" remain on the site
- [ ] Primary CTA copy is consistent across all placements
- [ ] CTA button styling is visually dominant on every page where it appears
